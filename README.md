## üìã About the Project

This is a Single Page Application (SPA) portfolio designed to mimic the **Visual Studio Code** interface. Instead of a standard scrollable page, users navigate through "files" in the explorer to view my skills, projects, and contact info. It uses HTML and CSS to simulate code syntax highlighting, providing an immersive experience for other developers.

### Key Features (Implemented)
*   **üîç IDE Simulation:** A realistic layout featuring a Sidebar, File Explorer, Tab System, and Editor Area.
*   **üé® Syntax Highlighting:** CSS classes manually crafted to look like C#, Python, C++, and JSON code.
*   **üì± Fully Responsive:** Adapts from a full desktop IDE view to a mobile-friendly tabbed interface.
*   **üåç Multi-Language Support:** Automatically detects the user's browser language to switch between English and Portuguese (PT-BR).
*   **‚ú® Visual Effects:** Includes a neural network particle background (Canvas), typewriter effects, and 3D tilt interactions on project cards.

Since this is a static website, you don't need a backend server or complex installation. Follow the steps below:

*   A web browser (Chrome, Firefox, Edge, etc.).

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/SHOUTEDI/Portfolio-Site
    ```

2.  **Enter the project folder:**
    ```bash
    cd Portfolio-Site
    ```

3.  **Run the project:**
    *   Simply double-click `index.html` to open it in your browser.
    *   **OR** if using VS Code, right-click `index.html` and select "Open with Live Server".
